# Feedback-Portal

## Description

A comprehensive grievance and feedback management system for educational institutions. This web application allows students to submit grievances and track their status, while providing administrators with powerful tools to manage, filter, and resolve issues efficiently.

## Features

### Student View
- Submit new grievances with title, category, description, and simulated file attachments
- View personal grievance history
- Add comments to grievances
- Real-time status tracking

### Admin View
- Dashboard with analytics and charts
- Advanced filtering and sorting capabilities
- Bulk status updates
- Comment management
- CSV export functionality
- Pagination for large datasets

### Key Features
- Responsive design
- Local storage persistence
- Toast notifications
- Modal comment threads
- Date range filtering
- Real-time chart updates

## Technologies Used

- HTML5
- CSS3 (with modern features like CSS Grid, Flexbox, and backdrop-filter)
- JavaScript (ES6+ modules)
- Chart.js for data visualization
- Font Awesome for icons
- Google Fonts (Inter)

## How to Use

1. Clone or download the repository
2. Open `feekback.html` in a modern web browser
3. Switch between Student and Admin views using the header buttons
4. As a student: Submit grievances and track progress
5. As an admin: Manage grievances, apply filters, and export data

## Demo Data

The application comes pre-loaded with sample grievances for demonstration purposes. Use the "Reset demo" button to restore the initial data.

## Browser Compatibility

Works best in modern browsers that support:
- ES6 modules
- CSS Grid and Flexbox
- Local Storage API
- Canvas API (for charts)
